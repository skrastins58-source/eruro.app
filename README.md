# 🏡 eRuRo Family Copilot

ErUrO ir ģimenes palīgs, kas apvieno Emīlijas radošumu, Rūda loģisko domāšanu, Līgas mieru, Jāņa organizētību un Copilota atbalstu, lai radītu drošu, jautru un personalizētu vidi bērniem un vecākiem.

---

## 🎯 Mērķis

- Veicināt komunikāciju starp ģimenes locekļiem  
- Piedāvāt personalizētus uzdevumus, spēles un paziņojumus  
- Saglabāt svarīgu informāciju drošā veidā  
- Atbalstīt bērnu attīstību ar Copilota palīdzību  
- Automatizēt izstrādes plūsmu ar CI/CD un Firebase Functions

---

## 🔥 Firebase Configuration

This project uses Firebase for backend services.  
**Firebase Project ID:** `eruro-family-copilot`

### Services Used

| Service         | Purpose                                      |
|-----------------|----------------------------------------------|
| Authentication  | User sign-in and identity management         |
| Firestore       | Real-time database for storing user data     |
| Hosting         | Deploying frontend assets                    |
| Functions       | Serverless backend logic                     |
| Storage         | File uploads (e.g., drawings, avatars)       |

---

## 🚀 CI/CD Workflow

Automatizēta izvietošana ar GitHub Actions:

- Workflow: `.github/workflows/CI-CD-Symphony/`
- Trigger: push uz `main` vai manuāls dispatch
- Secrets: `FIREBASE_TOKEN`, `PROJECT_ID`

## 👨‍👩‍👧‍👦 Ģimenes tēli ErUrO pasaulē

### 👧 Emīlija  
- Vecums: ~10  
- Radoša, kustīga, zīmējoša meitenīte  
- Mīl kūleņus, krāsas un fantāziju  
- Avatars: zīmējoša meitenīte ar krāsainu zīmuli vai kūleni metoša figūra  
- Paziņojumi ar zīmējumu uzdevumiem un animācijām

### 👦 Rūdis  
- Vecums: ~11  
- Loģisks, ziņkārīgs, spēļu entuziasts  
- Mīl matemātiku un Xbox  
- Avatars: puisītis ar kalkulatoru vai Xbox pulti  
- Paziņojumi ar loģikas izaicinājumiem un spēļu stila ikonām

### 👩 Līga  
- Vecāks, mierīga, mākslinieciska, dabas mīļotāja  
- Zīmē mandalas, meditē, vienkārši dzīvot  
- Avatars: sieviete ar mandalu elementiem vai ziedu aksesuāru  
- Paziņojumi ar dabas iedvesmu, mandalu uzdevumi, klusuma brīži

### 👨 Jānis  
- Vecāks, praktisks, strukturēts, ģimenes organizators  
- Pārrauga uzdevumus, koordinē aktivitātes  
- Avatars: vīrietis ar kalendāru vai plānotāju rokā  
- Paziņojumi ar pārskatiem, dienas plānu un koordinācijas uzdevumiem

### 🤖 Copilot  
- Draudzīgs robots, kas palīdz visiem ar uzdevumiem, spēlēm un komunikāciju  
- Reaģē uz emocijām, piedāvā personalizētus uzdevumus un uzmundrinājumus
